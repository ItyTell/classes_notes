\section{Третя задача}

\begin{tcolorbox}[title=Задача 3]
    \begin{center}
        Наближено розв'язати ЗКШ, використавши метод Ейлера з кроком 0.5:
        $$ y'' = xy - x, y(0)=y'(0)=2, x \in [0, 2]$$
    \end{center}
\end{tcolorbox}

$$\frac{dy'}{dx} = xy - x$$
$$\frac{dy}{dx} = y'$$
За методом Ейлера виходить:
$$y'(x+h) = (xy(x) - x)h + y'(x)$$
$$y(x+h) = y'(x)h + y(x)$$
$$
\begin{cases}
    y'(0)=2, y(0) = 2 \\
    y'(0.5)= 2, y(0.5) = 3\\
    y'(1) = 2.5, y(1) = 4\\
    y'(1.5) = 4, y(1.5) = 5.25 \\
    y'(2) = 7,1875, y(2) = 7.25
\end{cases}
$$
